== Description ==

PopuList allows you to track the popularity of your posts on social bookmarking sites reddit, stumbleupon, del.icio.us and digg. When PopuList is activated, a page is added to your dashboard giving the number of times a page has been saved or upvoted.

New in version 1.5 PopuList now also lets you view backlinks as indexed by Google and Yahoo.

== Installation ==

1. Upload the plugin (unzipped) into `/wp-content/plugins/`. You should end up with a `/wp-content/plugins/Populist/` directory containing the plugin files.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Dashboard and click the link for Populist in the menu to view your stats.

== Frequently Asked Questions ==

= Why do my StumbleUpon rankings never go higher than 5? =

All the other rankings are the number of times a page has been saved or upvoted. StumbleUpon however don't provide the exact figures, so the number is a rating out of 5 (5 means the page is VERY popular!)
